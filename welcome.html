歡迎使用 AWS Pricing Calculator 助手

讓 AWS 價格計算變得更簡單

關於此擴展程式

AWS Pricing Calculator 助手讓您能夠使用自然語言描述需求，自動在 AWS Pricing Calculator 上進行價格估算，無需手動填寫複雜的表單。

主要功能

  • 透過自然語言輸入直接估算 AWS 服務價格
  • 自動識別常見 AWS 服務的需求描述
  • 支援 EC2、RDS、S3 等多種 AWS 服務
  • 無縫整合到 AWS Pricing Calculator 頁面
  • 簡單直觀的使用者界面

開始使用

  1. 前往 AWS Pricing Calculator 網站
  2. 點擊瀏覽器工具列中的擴展程式圖標或頁面右下角的 AWS 助手按鈕
  3. 用自然語言描述您的 AWS 服務需求，例如「我需要 5 台 t3.medium EC2 實例和 1 個具有 100GB 儲存空間的 RDS MySQL 資料庫」
  4. 點擊「估算價格」按鈕，讓擴展程式自動填寫相應的表單
  5. 查看生成的價格估算，並可以進一步調整細節
